## よく使うショートカット

### 一括変換
- 変更したい変数の場所で```ctrl+F2```

### settings.jsonを開く
- ```ctrl+,```

### settings.json再読み込み
- コマンドパレットで```reload```, ウィンドウの再読み込みを選択

### コマンドパレットを開く
- ```ctrl+shift+p```
