

## 環境構築
- Anaconda入れた
- PATH通した
```bash:.bash_profile
ANA_PATH=/cygdrive/c/Anaconda3
export PATH=\$ANA_PATH:\$ANA_PATH/Library/bin:\$ANA_PATH/Scripts:$PATH
```


## Anaconda3コマンドまとめ

### よく使う
- 現在の仮想環境情報
```bash
conda info
```

- インストールされているパッケージ一覧表示
```bash
conda list
```
